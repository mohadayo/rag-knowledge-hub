# 経費精算 業務手順書

## 1. 概要
業務に関連する経費を精算する手順を説明します。経費精算は毎月締め切りがあるため、期限内に必ず申請してください。

## 2. 対象となる経費
- 交通費（電車、バス、タクシー）
- 出張旅費（宿泊費、日当）
- 接待交際費
- 書籍・研修費
- 消耗品購入費

## 3. 申請手順

### 3.1 経費精算システムへのアクセス
1. 社内ポータルから「経費精算システム」をクリック
2. 社員IDとパスワードでログイン
3. 「新規申請」ボタンをクリック

### 3.2 申請内容の入力
1. 経費の種類を選択
2. 利用日を入力
3. 金額を入力
4. 利用目的・内容を記載（具体的に）
5. 領収書の写真をアップロード

### 3.3 承認フロー
1. 申請者が申請を提出
2. 直属の上長が承認
3. 経費が5万円以上の場合は部長承認も必要
4. 経理部が最終確認
5. 翌月25日の給与と合わせて支給

## 4. 注意事項
- 領収書は必ず原本を保管してください（電子保存法対応済み）
- 申請期限：毎月末日まで（翌月5日が最終期限）
- タクシー利用は原則として終電後または荷物搬送時のみ
- 接待交際費は事前に上長の承認を得てください
- 1回あたり10万円以上の経費は事前申請が必要です
